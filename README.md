# daring-cyclops

Daring Cyclops is a [online multiplayer](https://en.wikipedia.org/wiki/Multiplayer_video_game#Networked) game inspired by [CompuServe](https://en.wikipedia.org/wiki/CompuServe) [MegaWars](https://en.wikipedia.org/wiki/MegaWars).  MegaWars is itself a clone of [Decwar](https://en.wikipedia.org/wiki/Decwar).

I greatly enjoyed MegaWars in the 1980s, and I have long wanted to write something similar. 

Daring Cyclops is written in [Go](https://en.wikipedia.org/wiki/Go_(programming_language)) and deploys via [Docker](https://en.wikipedia.org/wiki/Docker_(software)) in [Kubernetes](https://en.wikipedia.org/wiki/Kubernetes]).  There are additional dependencies on [RabbitMQ](https://www.rabbitmq.com/) and [PostGreSQL](https://www.postgresql.org/).

Big thanks to Merlyn Cousins for making [Decwar sources](https://github.com/drforbin/decwar) available.

Still a work in progress, visit later and hopefully I will have more to share.
